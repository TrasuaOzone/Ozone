1. base.css – Cài đặt nền tảng
2. layout.css – Bố cục trang
3. components.css – Thành phần giao diện
4. responsive.css – Tối ưu cho thiết bị
🔗 Cách liên kết trong HTML
Trong <head> của index.html, bạn thêm:

html
<link rel="stylesheet" href="css/base.css">
<link rel="stylesheet" href="css/layout.css">
<link rel="stylesheet" href="css/components.css">
<link rel="stylesheet" href="css/responsive.css">

global.css         → Biến màu, font, reset, cấu trúc chung
├── header.css         → Giao diện logo, tiêu đề, slogan, nút đăng nhập/đăng ký
├── card.css           → Giao diện các dịch vụ yêu thích, trà sữa, trái cây
├── form.css           → Giao diện form đăng nhập, nhập địa chỉ, trường input
├── popup.css          → Overlay, dialog, nút đóng, hiệu ứng mở
├── map.css            → Giao diện bản đồ, nút xác nhận, ghim vị trí
├── footer.css         → Chân trang, liên kết điều khoản, chính sách
├── nav.css            → Thanh điều hướng dưới cùng (bottom-nav)
├── responsive.css     → Media queries cho mobile, tablet, desktop

/js/
├── main.js           → Khởi tạo, liên kết các module
├── auth.js           → Đăng nhập, đăng xuất, quản lý phiên
├── ui.js             → Hiển thị popup, overlay, cập nhật giao diện
├── map.js            → Leaflet, định vị, reverse geocode
├── suggest.js        → Autocomplete địa chỉ
